Presenting an evaluation pipeline assessing the utility of synthetic data, for data sharing between organizations collaborating on AI development.

Synthetic data AI evaluation

Using two publicly available datasets, we developed a pipeline to generate synthetic data and evaluate its utility for training Machine Learning models. I compared model performance across Differential Privacy parameters and generation models. In order to present results to non-technical stakeholders, I created a demo dashboard to visualize and compare results across evaluation settings.
